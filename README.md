# Taskify 📝

Taskify is a simple and responsive task management application built using HTML, CSS, and JavaScript. It allows users to manage daily tasks efficiently with features like adding tasks, deleting tasks, marking tasks as completed, and persistent storage using localStorage.

---

## 🚀 Features

- ➕ Add new tasks
- ❌ Delete tasks
- ✅ Mark tasks as completed
- 💾 Save tasks in localStorage
- 🔄 Tasks persist even after page refresh
- 📱 Responsive and user-friendly interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- localStorage API
